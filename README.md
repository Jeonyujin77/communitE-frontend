# communitE-frontend

```
    communitE 프론트앤드 프로젝트
```

## 화면 기능
- 메인: /
- 회원가입: /join
-  로그인: /login
-  마이페이지: /mypage
-  마이페이지 수정: /mypagemodify
- 게시글 작성: /write
- 게시글 수정: /edit/:id
- 사용자별 게시글 조회: /post/:id

## 담당한 기능

- 이재정: 메인, 게시글 CRUD, 댓글 CRUD
- 전유진: 로그인/로그아웃, 회원가입, 마이페이지 조회, 마이페이지 수정
